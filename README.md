# Биом

**Биом** — это бесплатная система управления проектами с открытым исходным кодом, где проекты, задачи и команды образуют собственную экосистему.

## Быстрый старт

### Сборка

```bash
cmake --preset default
cmake --build --preset default
```

### Запуск тестов

```bash
ctest --preset default
```

## Структура проекта

| Директория      | Назначение                                    |
|-----------------|-----------------------------------------------|
| `vault-server/` | REST-сервер (cpprestsdk)                      |
| `client/`       | Кроссплатформенный клиент (Qt6/QML)           |
| `common/`       | Общий код, DTO, конфигурация, логирование     |
| `api/`          | Спецификации API (OpenAPI, доменные сущности) |
| `tools/`        | Мигратор БД и генераторы кода                 |
| `tests/`        | Модульные и интеграционные тесты              |

## Технологический стек

- **Язык:** C++23
- **Клиент:** Qt6, QML
- **Сервер:** cpprestsdk, nlohmann/json, jwt-cpp, OpenSSL
- **База данных:** SQLite
- **Тестирование:** Boost.Test
