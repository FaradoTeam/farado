# СХЕМА ПРОВЕРКИ ПРАВ

```
┌─────────────────────────────────────────────────────────────────┐
│                    isSuperAdmin = true?                         │
│                        Да → ДОСТУП РАЗРЕШЕН                     │
└─────────────────────────────────────────────────────────────────┘
                              │ Нет
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 1. Проверка RuleProject (проектный уровень)                │
│  ├── isReader? → для просмотра                                  │
│  ├── isWriter? → для создания/изменения элементов               │
│  ├── isProjectEditor? → для управления подпроектами             │
│  ├── isPhaseEditor? → для управления фазами (этапами, версиями) │
│  └── isBoardEditor? → для управления досками                    │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 2. Проверка RuleItemType (тип элемента)                    │
│  ├── isReader? → может читать элементы этого типа               │
│  └── isWriter? → может создавать/изменять элементы этого типа   │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 3. Проверка RuleState (переходы между состояниями)         │
│  └── isStateAllowed? → может перевести элемент в это состояние  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                       ДОСТУП ОПРЕДЕЛЕН                          │
└─────────────────────────────────────────────────────────────────┘
```

## Условные обозначения для матрицы прав

| Символ   | Значение                                   | Примечание                    |
|----------|--------------------------------------------|-------------------------------|
| **S**    | Супер-админ — доступ ко всему              | User.isSuperAdmin=1           |
| **R_R**  | Право на чтение проекта                    | RuleProject.isReader=1        |
| **R_W**  | Право на запись элементов проекта          | RuleProject.isWriter=1        |
| **R_PE** | Право на редактирование проекта            | RuleProject.isProjectEditor=1 |
| **R_PHE**| Право на редактирование фаз                | RuleProject.isPhaseEditor=1   |
| **R_BE** | Право на редактирование досок              | RuleProject.isBoardEditor=1   |
| **T_R**  | Право на чтение типа элемента              | RuleItemType.isReader=1       |
| **T_W**  | Право на запись типа элемента              | RuleItemType.isWriter=1       |
| **S_A**  | Право на переход в состояние               | RuleState.isStateAllowed=1    |
| **R_C**  | Право на создание корневого проекта        | Rule.isRootProjectCreator=1   |
| **A_U**  | Административные права на пользователей    | для MVP -- только Супер-админ |
| **A_R**  | Административные права на роли             | для MVP -- только Супер-админ |
| **A_W**  | Административные права на рабочие процессы | для MVP -- только Супер-админ |
| **A_T**  | Административные права на типы элементов   | для MVP -- только Супер-админ |
| **A_C**  | Административные права на календарь        | для MVP -- только Супер-админ |
| **U_O**  | Пользователь = владелец объекта            |                               |
